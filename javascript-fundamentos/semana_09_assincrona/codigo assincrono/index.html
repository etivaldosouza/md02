Código Assimcrono

  • Leva um Tempo para ser executado, e pode ser bem sucedido ou não. ou seja, pode ter erro ou não

    • Ex: Chamadas para APIs, Iterações com Banco de Dados etc.

  • Há 3 Maneiras de Lidar com esse tipo de código:
    1. Callbacks
    2. Promises()
    3. Promises com Async/Await
