API

"Aplication Progame Interface"

uma forma de você se comunicar com programas

Boas Praticas:

  1. Utilizar verbos HTTP
  2. ñ utilizar barra no final
  3. se usar o Resource no plural os demais que vc for utilizar deve ser tb no plural

Verbos HTTP

  • GET: Receber dados de um Resource
  • POST: Enviar dados p/ ser processadas por um Resource
  • PUT: Atualizar dados de um Resource
  • DELETE: Apagar um Resource
